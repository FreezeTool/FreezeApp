# FreezeApp
项目最早使用了WebSocket通信，后来发现[Shizuku](https://github.com/RikkaApps/Shizuku)的原理更简单。因此复制了[Shizuku](https://github.com/RikkaApps/Shizuku)中的部分代码。

## 已有能力
* APP冻结管理
* APP电池记录
* App使用记录
* APP权限管理
* APP实时监控
* APP存储管理
* APP粘贴板管理
* APP待机分桶
* APP休眠模式（白名单配置）
* APP外屏配置（XIAOMI MIX FLIP）
